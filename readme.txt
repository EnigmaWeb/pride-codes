A simple plugin to help show your support for your LGBTQI+ colleagues.

== Description ==

[pride.codes](https://pride.codes/) provides a collection of simple widgets for businesses to add to their sites. We want businesses to actively show their support for their LGBTQI+ colleagues


== Installation ==

1. Upload the 'pride-codes' folder to your '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the 'Settings > Pride Codes' menu to update the plugin settings


== Frequently Asked Questions ==

= What is this for? =
[pride.codes](https://pride.codes/) is a collection of simple web widgets that allow you to show your support for marriage equality and the LGBTIQ+ community. We want to help technologists worldwide fly the rainbow flag and make the internet a brighter place.

= Who wrote the script? =
The scripts that this plugins loads were developed by Australian developer, [Jack Skinner](https://developerjack.com/). All the code is MIT licensed and can be found on [Jack's Github page](https://github.com/devjack/pride.codes/).
